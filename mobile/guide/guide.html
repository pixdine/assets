Ttypography

  • 국문 영문 모두 Pretendard 폰트를 사용하며, 폰트 굵기는 font-weight로 구분한다.
  • 폰트 속성은 Regular, Medium, Semibold 3가지만 사용한다.
  • 기본 설정은 {font-family: Pretendard, "malgun gothic", "맑은 고딕", sans-serif; font-weight:400;}

400 - Pretendard Regular 가나다라마바사아자차카타파하

500 - Pretendard Medium 가나다라마바사아자차카타파하

600 - Pretendard Semibold 가나다라마바사아자차카타파하

Color

  • color-primary와 color-secondary는 엠티처에서 사용하는 변수명에 컬러값을 재정의함.
  • #5177FF--color-primary
  • #5177FF--color-primary-150
  • #3959CC--color-secondary
  • #19244D--color-tertiary

						--color-primary: #5177FF;

						--button-primary-disable: #F1F1F5;
						--button-secondary: #19244D;
						--button-secondary-hover: #091B58;

Popup (엠티처와 마크업 동일)

  • target-obj의 값을 원하는 팝업의 id 값과 동일하게 넣어주면 팝업 실행됩니다.
  • init-show 라고 속성값을 붙이면 모달 초기값이 show 됩니다.
  • 클릭이벤트 없이 팝업 및 알럿 실행해야 하는 경우 : 아래 함수를 실행시켜 주면 됩니다.
    openPopup({id: "모달or알럿 아이디"});
  • 특정 조건 이후 팝업&알럿이 닫혀야 하는 경우 : 아래 함수를 실행시켜 주면 됩니다.
    closePopup("모달or알럿 아이디");
